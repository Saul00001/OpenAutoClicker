OpenAutoClicker

OpenAutoClicker es un auto-clicker simple en desarrollo que está abierto a colaboraciones.

Versión: 0.01

 Características
- Fácil de usar
- Configurable para diferentes intervalos de clic
- Código abierto y personalizable

 Requisitos
- Python 3.x
- PyQt5 (para la interfaz gráfica)
